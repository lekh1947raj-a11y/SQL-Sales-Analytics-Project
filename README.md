# SQL Sales Analytics Project

## Overview
This project performs end-to-end sales analytics using MySQL.
It covers sales performance, customer behavior, product analysis,
regional insights, and return/refund analysis.

## Database Schema
The schema below represents the relationships between customers,
orders, order details, products, and regions.

[Database Schema](Schema.png)

## Files in this Repository
- sales_analysis.sql  
  → Contains all analytical SQL queries (sales, customers, products, trends)

- procedures.sql  
  → Contains stored procedures (e.g., order volume by region)

- Schema.png  
  → ER / database schema diagram

## Key Insights Covered
- Total & monthly revenue analysis
- Average Order Value (AOV)
- Top customers & repeat customer rate
- Product & category performance
- Regional sales & order volume
- Return & refund analysis

## Tools Used
- MySQL
- SQL (Joins, CTEs, Window Functions, Stored Procedures)
